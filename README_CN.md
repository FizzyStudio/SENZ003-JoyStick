### 产品介绍

> 采用原装优质金属PS2摇杆电位器制作，具有(X,Y)2轴模拟输出，(Z)1路按钮数字输出。配合Arduino传感器扩展板可以制作遥控器等互动作品。

### 技术参数

* 接口类型：模拟
* 三轴（X，Y，Z（按钮））
* PH2.0接口
* 尺寸：37x25x32mm
* 重量：15克

### 应用
* 游戏手柄
